Odoo Website Sorting and Shopping

Odoo plugin provides sorting and shopping interface on eCommerce.

  • This Extension Include

    • Provides new interface like: Shop By Product, Shop By Category, and Shop By Brand
    • Supports product sorting on the basis of there attributes.
    • Options to assign image and description on the brands.
    • Options to assign image and description on the categories.
    • Sorting attributes are customizable.
    • Admin can create multiple combinations of sorting criteria.
    • Admin can enable and disable the shop by and sort by options.
    • Admin can enable and disable the brand and category description.

  • Setup & Configure

    • This module required the extension of "Website Product Brands" module.

    • After successfully installation of the module, you can locate the menu "Website Sorting" inside "Sale" menu.
    • Here you can create and manage attributes for sorting of your products.

  • Website Shopping

    • Shop By Product

    • Shop By Brand: Here the user can view the list of brands with there image and description, and also filter the products related to these brands.

    • Brand Image/Description: The brand image and description will visible on the top of filtered products.

    • Shop By Category: A User can also filter the products from the "Shop By category" page.

    • Category Image/Description: Category banner and description will visible on the top products filtered from the "Shop By Category" page

  • Website Sorting

    • On your website, the option of "Refine By" will become available.
    • You can set the refine by parameters from the backend.

    • Sort By Fresh Arrivals

    • Sort By Pricing
